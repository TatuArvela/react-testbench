# react-testbench

## client

React project base with the following flavors:

* TypeScript
* react-router-dom
* styled-components
* ESLint with:
    * Prettier
    * simple-import-sort
* Jest and Testing Library with:
    * jest-styled-components
    * msw

Structured into:
* `app`, `auth`, `views`
